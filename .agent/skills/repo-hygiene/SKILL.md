# SKILL: Repo Hygiene (GitHub-ready)
## Goal
Repo pulito e sicuro.
## Regole non negoziabili
- .env in .gitignore; creare .env.example senza segreti.
- Niente token/chiavi/password nel codice.
- README minimo: run locale, env vars, migrazioni, test.
## Output richiesto
- .gitignore additions, .env.example template, README sections, CI base.
